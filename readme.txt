== Description ==

This Plugin lists the latest jobs for web developers in a page or post (via shortcode) from web-engineering-jobs.com.

Please report bugs to kontakt@webworks-nuernberg.de

Visit <a href="http://www.web-engineering-jobs.com">www.web-engineering-jobs.com</a> for more informations.

== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress.
2. Activate the plugin at the plugin administration page
3. Putt the shortcode "[web-engineering-jobs]" into any page(s) or post(s)
4. Done
